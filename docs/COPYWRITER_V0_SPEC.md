# Copywriter v0 Specification

Copywriter v0.1 is the first real AI employee for AI Business OS.

It should prove that the Client Template can create business value before broader media automation is built.

## Capability

Copywriter v0.1 creates marketing copy from Business DNA and sends selected output to Approval Center.

## Required Inputs

- Business name.
- Services or products.
- Target audience.
- Brand voice.
- Primary offer.
- Primary CTA.
- Languages.
- Restrictions.
- Marketing goal.
- User request.

## First Workflow

1. User opens Copywriter.
2. User chooses content type: social post, email draft, ad copy, or landing page section.
3. User enters a short request.
4. Copywriter reads Business DNA.
5. Copywriter creates three draft options.
6. User selects one draft.
7. User can request edits or send to Approval Center.
8. Approval item is created.
9. User approves the item.
10. Approved content is saved.
11. Activity events are recorded.

## Draft Output

Each draft should include:

- Draft ID.
- Title.
- Body.
- Suggested CTA.
- Channel.
- Tone.
- Business DNA version.
- Prompt package version.
- Created date.

## Validation Level

Copywriter social post draft should reach Level 5 before being sold as a package promise.

Other copy types may start as Level 2 or Level 3.

## Not Included Yet

Copywriter v0.1 should not:

- Publish automatically.
- Generate images.
- Generate video.
- Manage paid ads.
- Replace human approval.
